## 安装使用

- 获取项目代码

```bash
git clone https://github.com/buqiyuan/vue3-antdv-admin
```

- 安装依赖

```bash
cd vue3-antdv-admin

pnpm install

```

- 运行

```bash
pnpm dev
```

- 打包

```bash
pnpm build
```
